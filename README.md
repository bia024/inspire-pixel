# InspiraPixel

## Author

Bianca Caetano, Software Engineering Student

InspiraPixel is a Vue 3 application for visual inspiration, showcasing a gallery of landscape and architecture images. Users can explore images, favorite their preferred ones, and search by title or category.

## Features

- Image gallery with category filters (nature, architecture)
- Favorites system with local persistence
- Search images by title or category
- Responsive and accessible design
- Intuitive navigation

## Technologies Used

- Vue 3 (Composition API)
- Vite
- Sass/SCSS
- Iconify (icons)
- Vue Router
- Vitest (unit testing)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/inspirapixel.git
   cd inspirapixel

   ```

2. Install dependencies:

npm install

3. Run the project in development mode:

npm run dev

4. Open http://localhost:5173
   in your browser.

## Usage

Browse the image gallery

Click the heart to favorite/unfavorite images

Use the search bar to filter images by title or category

Switch between "All Images" or "Favorites" tabs to view specific sets

## Available Scripts

npm run dev: Starts the development server

npm run build: Builds the project for production

npm run preview: Preview the production build

npm run test: Runs unit tests

# InspiraPixel

Um aplicativo Vue 3 para inspiração visual, apresentando uma galeria de imagens de paisagens e arquitetura. Permite aos usuários explorar imagens, favoritar suas preferidas e buscar por título ou categoria.

## Funcionalidades

- Galeria de imagens com filtros por categoria (natureza, arquitetura)
- Sistema de favoritos com persistência local
- Busca de imagens por título ou categoria
- Design responsivo e acessível
- Navegação intuitiva com Vue Router

## Tecnologias Utilizadas

- Vue 3 (Composition API)
- Vite
- Sass/SCSS
- Iconify (ícones)
- Vue Router (roteamento)
- Vitest (testes unitários)

## Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/inspirapixel.git
   cd inspirapixel
   ```

2. Instale as dependências:

   ```bash
   npm install
   ```

3. Execute o projeto em modo de desenvolvimento:

   ```bash
   npm run dev
   ```

4. Abra [http://localhost:5173](http://localhost:5173) no navegador.

## Uso

- Navegue pela galeria de imagens
- Clique no coração para favoritar/desfavoritar imagens
- Use a barra de busca para filtrar imagens por título ou categoria
- Acesse as abas "Todas as Imagens" ou "Favoritas" para alternar visualizações
